Tipos de software
Software de sistema
Programas que dan al usuario la capacidad de relacionarse con el sistema, para ejercer control sobre el hardware. 		Software de aplicación
Programas diseñados como herramientas que le permiten a un programador desarrollar programas informáticos. 		Software de programación
Programas diseñados para realizar una o más tareas específicas a la vez, pueden ser automáticos o asistidos.
Software libre
El software libre es un software en el que los usuarios pueden acceder a su código fuente para estudiarlo, copiarlo o modificarlo. 		Software propietario
En software propietario su código fuente no está disponible al público y cuya licencia de uso está controlada. Generalmente, se debe pagar y su modificación o redistribución no está permitida.
Ejemplos

Visual Studio Code es un software para programar Visual Studio Code
Android es software de sistema Operativo Movil Android
Windows es software un sistema Operativo para PC Windows
Minecraft es un software de aplicacion, es un videojuego Minecraft
